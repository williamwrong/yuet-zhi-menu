# 肉碎煮豆腐

### Youtube
[![thumbnail](https://i.ytimg.com/vi/CGF_-9SNhoQ/hq2.jpg)](https://www.youtube.com/watch?v=CGF_-9SNhoQ)

### 材料
1. 百福蒸煮豆腐一盒
2. 豬肉碎
3. 蔥花

### 調味料
1. 生抽 1湯匙
2. 老抽 2.5湯匙
3. 水 1碗
4. 冰糖適量（7-8克）

### 生粉芡
1. 生粉 1.5茶匙
2. 水 2湯匙

### 步驟
1. 先準備調味料同生粉芡
2. 豆腐切粒
3. 煮肉碎
4. 7成熟落調味料
5. 煮熟肉碎後加豆腐
6. 煮2分鐘
7. 加入生粉芡
8. 加上蔥花
